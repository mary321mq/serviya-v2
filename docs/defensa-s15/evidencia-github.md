# Evidencia GitHub

La rubrica exige que el aporte individual sea verificable en GitHub y este integrado en la rama comun.

## Evidencia grupal

| Evidencia | Donde mostrarla |
| --- | --- |
| Rama integrada evaluada | Branch principal o rama final U3. |
| Commits del equipo | Historial de commits. |
| Pull Requests o merges | Seccion Pull Requests o historial Git. |
| Resolucion de conflictos | Commits de merge o capturas. |
| Documentacion | Cambios en `docs/` y `mkdocs.yml`. |

## Evidencia individual

Cada integrante debe tener un anexo con:

- Nombre.
- Rol o responsabilidad.
- Rama de trabajo.
- Commits de codigo.
- Commits de documentacion.
- Pull Requests o merges.
- Evidencia de la demo en vivo.
- Evidencia de integracion en la rama comun.

## Comandos utiles

```powershell
git log --oneline --decorate --graph --all -20
```

```powershell
git shortlog -sn
```

```powershell
git log --author="NOMBRE" --oneline
```
